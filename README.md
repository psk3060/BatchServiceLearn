### 1. 목적
<code>
	스프링 배치 학습용
</code>

### 2. 연동
<ul>
	<li>Spring Batch</li>
	<li>MariaDB</li>
	<li>MyBatis</li>
	<li>Redis, JMS,  : 별도의 URL로 메시지 전달할 때 사용</li>
</ul>

### 3. 기능

<ol>
	<li>휴면계정 전환 : 별도의 URL로 전달(휴면 계정으로 전환하고, 별도의 리턴값 전달)</li>
	<li>휴면계정 안내 : 안내용 이메일 보관은 배치 자체에서 처리</li>
</ol>
